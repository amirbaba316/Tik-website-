tik

Tik is a Django Powered Website built by Amir Baba and Abrar Wali. In This Website you can Lookup problems and solve them using your favourite language and submit the correct answer. The concept of this module was taken from Project Euler. In addition to problem solving tik also provides the discussion facility for users. Discussion and Problems module can be viewed by anyone but in order to sumbit the answer and to use discussion, user must be registered on the website. This website serves project-based learning approch.
